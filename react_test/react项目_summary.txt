React.StrictMode 组件是什么？
StrictMode 是一个用来检查项目中潜在问题的工具。它为其后代元素触发额外的检查和警告。
    1、识别不安全的生命周期
    2、关于使用过时字符串 ref API 的警告
    3、关于使用废弃的 findDOMNode 方法的警告
    4、检测意外的副作用
    5、检测过时的 context API
与 Fragment 一样，StrictMode 不会渲染任何可见的 UI。
注意：严格模式检查仅在开发模式下运行；它们不会影响生产构建。
